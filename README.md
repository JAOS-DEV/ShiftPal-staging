# ShiftPal

ShiftPal is a modern, responsive web application designed for professional drivers in the UK. It serves as a digital assistant to help track work hours, calculate pay, monitor compliance with legal driving limits, and provide instant advice from an AI union representative.

## 🚀 Production Environment

This is the **production environment** for ShiftPal.

- **Live URL**: `https://jaos-dev.github.io/ShiftPal/`
- **Status**: Live and serving customers
- **Firebase Project**: Production environment

## 📚 Documentation

- **Staging Environment**: See `README-staging.md` for development information

## 🔧 Quick Start

```bash
# Clone the repository
git clone https://github.com/JAOS-DEV/ShiftPal.git
cd ShiftPal

# Install dependencies
npm install

# Start development server
npm run dev
```
